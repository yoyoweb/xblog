# X-Blog

Blog with X-Tag.

Use X-Tags library to create web components useful to create a blog:

 - `x-blog` read a remote YAML config file, containing titles and URLs of posts;
 - `x-blog-posting` read and display an article from a remote file formatted in YAML and Markdown.
